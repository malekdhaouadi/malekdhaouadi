## Selected Projects

### MicroSure — Fintech Inclusion Platform
> Angular 18 · Spring Boot · MySQL · Redis · LLM (Groq/LLaMA) · Docker

A production-ready microfinance platform replacing physical collateral with community reputation scoring. Features 7 integrated modules including digital ROSCAs, parametric insurance, an agentic AI advisor, and fraud detection via SVM, XGBoost, and Isolation Forest. Financial logic includes EMI computation, actuarial scoring, and Monte-Carlo portfolio simulation.

---

### Palm Vein Biometric Recognition System
> Python · PyTorch · ResNet18 · OpenCV · scikit-learn

Deep learning pipeline for identity authentication based on palm vein patterns. Implements transfer learning with ResNet18 pre-trained on ImageNet, fine-tuned on 1,200 images across 50 individuals. Outperforms classical ML baselines (SVM, Random Forest) on precision, recall, and F1-score.

---

### Physioform — Intelligent Healthcare Desktop Application
> C++ · Qt · Python · Arduino · OpenCV

Full-featured clinic management system integrating motion detection, facial recognition, voice processing, and an interactive chatbot. Automated scheduling, payments, and administrative notifications — designed for non-technical end-users.
